# Design（技术设计文档）

## 职责

定义系统"怎么做"——基于 PRD 产出技术方案。

## 存放内容

- **CHANGELOG.md**：所有技术变更记录
- **架构设计**：系统架构图、模块划分、技术选型
- **数据模型**：数据库 ER 图、表结构、Form Schema 定义
- **API 契约**：OpenAPI 规范（openapi.yaml），前后端接口契约
- **时序图/流程图**：关键业务流程的交互设计

## 边界

| 是 | 不是 |
|----|------|
| 技术架构、数据模型、接口定义 | 产品需求、用户场景 |
| 怎么做、用什么技术、怎么交互 | 要做什么、为什么做 |
| 前后端契约 | 测试用例 |

## 负责人

架构师
