# References（业务参考资料）

## 职责

存放原始业务政策和法规文件，作为 PRD 和 Design 的事实依据。

## 存放内容

- 政府部门的材料提交说明
- 行业管理制度模板
- 政策法规原文
- 其他业务方提供的原始文件（Word、PDF 及其 Markdown 转换版）

## 边界

| 是 | 不是 |
|----|------|
| 原始业务文件（只读，不修改） | 产品需求、技术设计 |
| 事实依据，引用来源 | 被引用的产物 |
| 按业务类型分子目录存放 | 按阶段或角色存放 |

## 使用方式

- PRD 编写时引用此处的材料清单作为功能依据
- 文件内容不可修改，如需更新应替换整个文件并记录来源
- 支持 Word、PDF 原始格式及对应的 Markdown 可读版本

## 负责人

产品经理（收集和维护）
